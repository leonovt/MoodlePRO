3. Course-wide scoped summary — POST /courses/summary

Scope picker on the dashboard maps to: everything (no filter) / assignments (item_type == "assignment") / lectures (item_type == "lecture"). Each scope is summarized server-side after the extension re-fetches and scrapes the course page. Empty-result case is handled explicitly (no error):

scope=assignments, but the course has none → "No matching items were found for the requested scope."
